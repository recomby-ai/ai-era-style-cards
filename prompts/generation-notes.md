# Generation Notes

The current cards were generated as AI-era poster style comparison boards.

## Base Prompt Pattern

```text
Generate a wide AI-era poster style color-card comparison board with 6 panels on a black gallery background. Each panel is labeled with a large clean number and English style name only. Use the same subject across all panels: AI education, autonomous agents, data networks, future classroom, human judgment. Each panel contains 4 mini poster thumbnails in a 2x2 grid. Original designs, no logos, no watermarks, no real brands.

Batch XX styles:
1 Style Name: style-specific visual description.
2 Style Name: style-specific visual description.
...

Professional visual style atlas, strong differences, readable panel labels, abstract microtext acceptable.
```

## Production Advice

- Use AI-generated text only as visual texture.
- Replace all titles and microcopy with real typography before final publication.
- Keep each batch focused on the same topic so differences come from style, not subject matter.
- For future releases, generate one clean thumbnail per style in addition to the multi-style comparison cards.
- Use compressed thumbnails for gallery browsing and keep original PNG files as downloadable source images.
